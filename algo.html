This is the webpage for two JavaScript functions.

  1. That takes a single parameter and converts it to celsius, using a function named "convertFahrToCelsius".
  2. That takes a number, n, as an argument, creates an array of numbers from 1 to n and replaces multiples
    of 2, 3, and 5 with "yu", "gi" and "oh", then returns the resulting array, using a function named "checkYuGiOh".
